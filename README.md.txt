## Descrição

O LAS MDM é uma aplicação voltada para controle e gerenciamento de dispositivos móveis, com funcionalidades planejadas para uso interno da LAS Transportes.

---

## Como Usar

1. Instale o app em um dispositivo com Android 8.0+.
2. Ao abrir o app, insira a senha de acesso fornecida.
3. Após a autenticação, a mensagem de autorização será exibida.
